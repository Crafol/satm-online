get /map/regions
Get the regions map html file and either select starting location or plan movement to target site. This functionality requires the map to be opened in an IFRAME, because it will post a message to the parent window (see renderer.js for full details) (mapRegionsGet)

Query parameters

code (optional)
Query Parameter — If set, it is used as the company's starting location and the movement to the target site can be planned. If not set, the map allows to pick the starting location and closed thereafter.

Produces

  • text/html

Responses

200

Html file
