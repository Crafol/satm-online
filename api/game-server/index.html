Authentication

Up 
post /authenticate
Authenticate to the game. (authenticatePost)
This endpoint will reply with /authenticate/success only if the login is successful. Otherwise, it will say nothing and wait. With each new connection, the client has 3 seconds to send this authentication. Otherwise, the connection will be closed by the server anyway.

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body (required)
Body Parameter — Login data

Responses

204

This endpoint will reply with /authenticate/success only if the login is successful.
